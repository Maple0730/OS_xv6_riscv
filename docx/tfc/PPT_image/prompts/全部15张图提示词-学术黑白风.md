# xv6 进程管理课程设计 — 15 张配图提示词合集（学术黑白风格，v3 基于实际代码）

> **统一风格规范**（适用于全部 15 张图）
>
> - **配色**：纯黑白。深灰 #000000（主线条/文字）、中灰 #555555（次要线）、浅灰 #999999（辅助线/网格）、极浅灰 #DDDDDD（背景区块）、白色 #FFFFFF（背景）。禁止使用任何彩色。
> - **填充模式（区分系列用）**：白色实心 / 黑色实心 / 横线填充 `////` / 竖线填充 `\\\\` / 点阵 `....` / 斜线交叉 `xxxx` / 棋盘 `<>`。matplotlib 用 `hatch`。
> - **线型区分**：实线 / 虚线 `--` / 点线 `..` / 点划线 `-.`。
> - **字体**：英文/数字 Times New Roman（粗体 14pt 图题/12pt 轴/10pt 数值）；中文 宋体 SimSun（粗体 14pt 图题/12pt 标签/10pt 注释）。所有图统一 Times 风格。
> - **线宽**：主框 1.5pt、辅助 1.0pt、网格 0.5pt。
> - **排版**：单栏图（宽 8cm 或 16cm）、图题在图下方"Figure N. 标题"、子图用 (a)(b)(c) 标注。
> - **数学符号**：$E=mc^2$ 风格 LaTeX 渲染。坐标轴标签用 LaTeX 数学，如 $T_{\text{swtch}}(\mu s)$。
> - **图表元素**：图例放图内右上方或图下方，字号 9pt；数值标签直接标在节点/条形旁；网格仅水平/垂直细线，浅灰 #CCCCCC。
> - **比例**：单图 16:9 或 4:3；复合图用 1×2 或 2×2 子图。
> - **禁止**：阴影、渐变、3D 效果、emoji、彩色高亮；强调一律用加粗 + 边框加粗。
>
> **本文件结构**：风格规范 → F1-F5（段 1 进程生命周期）→ F6-F10（段 2 调度）→ F11-F15（段 3 同步与死锁）
>
> **代码来源说明**：所有图表数据均来自实际内核代码，图注中标注代码行号供答辩引用。

### F3 — fork / exec / exit / wait 完整流程图

**图表用途**：段 1 第 3 页，进程生命周期四大系统调用的完整内核流程。

**代码来源**：`kernel/proc.c` 第 406-484 行 `kfork()`；`kernel/exec.c` 第 27-151 行 `kexec()`；`kernel/proc.c` 第 528-567 行 `kexit()`；第 571-616 行 `kwait()`；第 488-522 行 `reparent()`

**数据**：4 段流程（fork→exec→exit→wait），双泳道（用户态/内核态），每段标出关键函数调用链。

**提示词正文**：

> 绘制学术黑白风格的双泳道流程图 Figure 3. fork / exec / exit / wait 完整流程。
>
> 整图 16:9。两个纵向泳道：左侧"User"（白色底）+ 右侧"Kernel"（**极浅灰 #EEEEEE 底**）。
>
> 四段流程（纵向堆叠，每段 2-4 个矩形框 + 箭头）：
>
> **fork 段**：
> - User: `fork()` → 子进程返回 0
> - Kernel: `kfork()` → `allocproc()` → `uvmcopy()`（白色）→ 复制 trapframe → 复制 fd → 加入 parent 链表 → state=RUNNABLE
>
> **exec 段**：
> - User: `exec(path, argv)` → 新程序
> - Kernel: `kexec()` → `namei()` → ELF 验证 → `proc_pagetable()` → 遍历 LOAD 段 `uvmalloc()` + `loadseg()` → 分配用户栈 → 原子切换页表 → 释放 old pagetable
>
> **exit 段**：
> - User: `exit(status)`
> - Kernel: `kexit()` → 关闭文件 → `reparent()`（将子进程转交 initproc）→ wakeup(parent) → state=ZOMBIE → `sched()`
>
> **wait 段**：
> - User: `wait(&status)` → pid
> - Kernel: `kwait()` → 扫描 proc 表找 ZOMBIE 子 → `freeproc()` 释放 trapframe/页表；未找到则 `sleep()` 等待
>
> 关键优化点用**粗 2pt 黑边 + 横线 `////` 填充**标注：`uvmcopy()`（全量复制）注释 "全量页复制，非 COW"；`reparent()` 注释 "僵尸回收关键"。
>
> 横向箭头 1pt 黑，垂直箭头 1.5pt。底部小字 9pt Times："fork≈450μs / exec≈1200μs（QEMU 单核实测）"。
>
> 代码标注（小字 8pt Times 斜体）："kfork: proc.c:406-484; kexec: exec.c:27-151; kexit: proc.c:528-567; kwait: proc.c:571-616"。
>
> 整图 16cm 宽。
