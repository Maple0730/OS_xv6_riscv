# 韩硕个人 PPT Mermaid 图源码

这一组文件对应个人 PPT 中最适合用结构图表示的 5 页：

- 第 3 页：系统启动整体框架
- 第 6 页：内存系统整体框架
- 第 8 页：内存系统扩展部分
- 第 10 页：文件系统整体框架
- 第 12 页：文件系统扩展一（日志 + 双盘）

## 文件对应关系

- `p3_system_boot_flow.mmd`
- `p6_memory_overview.mmd`
- `p8_memory_extensions.mmd`
- `p10_filesystem_layers.mmd`
- `p12_log_and_dualdisk.mmd`

## 使用方式

1. 直接复制到 Mermaid Live Editor 渲染
2. 导出为 `PNG` 或 `SVG`
3. 放入 PPT 后，再在 PPT 中补充少量颜色高亮或说明箭头

## 风格建议

- 背景：白色
- 主色：深蓝
- 辅助色：灰色
- 扩展功能节点可用浅橙色或浅蓝色强调

## 备注

这些图的目标是：

- 结构清楚
- 文字准确
- 便于课堂答辩讲解

因此没有追求复杂装饰，而是优先保证逻辑正确。
